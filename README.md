## Local Instalation

- run `git clone https://github.com/PROJECT-PIKSI-GANESHA-SIM/SIM-DOS.git`
- run `composer update`
- copy .env.example to .env
- run `php artisan key:generate`
- setting database di .env
- run `php artisan migrate --seed`
- run `php artisan serve`
